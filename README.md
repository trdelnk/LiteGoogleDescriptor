Add it in your root build.gradle at the end of repositories:
```
repositories {
  ...
  maven { url 'https://jitpack.io' }
}
```

Add the dependency
```
dependencies {
  implementation 'com.github.trdelnk:lite_google_descriptor:$version'
}
```
